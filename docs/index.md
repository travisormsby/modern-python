# Introduction

Python is over 35 years old, but there are ongoing significant changes in the ecosystem that affect best practices for writing Pythonic code. This workshop highlights a few of the most impactful changes from the past several years that you should get you rethinking your code:

- [Manage environments](01-manage-environments.md) with `uv` and `pixi`.
- [Create dynamic text](02-create-dynamic-text.md) with f- and t-strings.
- [Manipulate filepaths](03-manipulate-filepaths.md) with `Pathlib`.
- [Structure data](04-structure-data.md) with Dataclasses and Pydantic models.
- [Upgrade Python](05-upgrade-python.md) to take advantage of significant performance improvements
